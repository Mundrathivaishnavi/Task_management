# Team Task Manager

## Overview

Team Task Manager is a full-stack MERN application developed to manage projects, tasks, and team collaboration efficiently. The system provides role-based access where admins can create projects and tasks, assign team members, and monitor project progress, while members can view and update assigned tasks.

This project is designed with a modern responsive UI, animations, and secure authentication using JWT.

---

# Features

## Authentication
- User Signup
- User Login
- JWT Authentication
- Role-Based Access Control

## Dashboard
- Total Tasks
- Completed Tasks
- Pending Tasks
- Overdue Tasks
- Personalized User Dashboard

## Project Management
- Admin can create projects
- Assign team members
- Add project description and timeline
- Track project completion percentage

## Task Management
- Admin can create tasks
- Assign tasks to members
- Set task status
- Members can update task progress

## Team Members
- View assigned members
- View member tasks
- View assigned projects
